# Token Theft Investigation

## Objective

Determine whether authentication tokens have been compromised or abused.

## Investigation Steps

1. Review sign-in logs.
2. Investigate session anomalies.
3. Review conditional access results.
4. Examine authentication token usage.
5. Identify unusual locations or devices.

## Containment

- Revoke user sessions.
- Reset credentials.
- Require MFA authentication.

## Closure Criteria

- Sessions revoked.
- No further suspicious activity identified.
- Account secured.
