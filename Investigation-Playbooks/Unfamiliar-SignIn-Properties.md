# Unfamiliar Sign-In Properties Investigation

## Objective

Determine whether authentication activity differs from established user behavior.

## Investigation Steps

1. Review sign-in location.
2. Review device details.
3. Verify browser information.
4. Check authentication methods used.
5. Compare with historical sign-ins.
6. Validate with the user.

## Containment

- Revoke sessions.
- Reset credentials if required.
- Enforce MFA validation.

## Closure Criteria

- Activity validated.
- No indicators of compromise detected.
