# Global Administrator Activity Investigation

## Objective

Validate high-risk administrative activities performed by Global Administrators.

## Investigation Steps

1. Review audit logs.
2. Identify performed action.
3. Verify business requirement.
4. Review associated sign-ins.
5. Check source device.
6. Assess impact.

## Containment

- Revoke sessions if required.
- Disable account if compromise suspected.

## Closure Criteria

- Activity validated.
- Risk assessed.
- No unauthorized activity identified.
