# Azure Role Assignment Investigation

## Objective

Validate Azure RBAC role assignments and identify unauthorized privilege grants.

## Investigation Steps

1. Review assigned role.
2. Identify approving authority.
3. Review business justification.
4. Review assignment scope.
5. Check audit logs.

## Containment

- Remove unauthorized role.
- Review privileged actions.

## Closure Criteria

- Assignment validated.
- Approval confirmed.
- No suspicious activity identified.
